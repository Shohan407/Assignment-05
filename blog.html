Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where you add an event listener to the parent element, but use it to handle events on child elements. This eliminates the need to add event listeners to each child separately!

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event occurs on a child element in the DOM, it flows upwards. That is, the event flows from the bottom up (child - parent - grandparent - document).
